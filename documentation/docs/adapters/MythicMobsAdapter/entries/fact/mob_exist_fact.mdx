# Mob Exist Fact

A [fact](/docs/facts) that represents whether a MythicMobs mob is in the world.

<ReadonlyFactInfo />

## How could this be used?

This fact could be used to change dialogue sent by an NPC or mob when a boss exists. It could also be used in conjunction with the Spawn Mob action to spawn specific mobs if a mob exists/doesn't exist.

## Fields

<FactsField />

<EntryField name="Mob" required>
    The mob's name used to check.
</EntryField>
