# Spawn Mob Action

The `Spawn Mob Action` action spawn MythicMobs mobs to the world.

## How could this be used?

This action could be used in a plethora of scenarios. From simple quests requiring you to kill some spawned mobs, to complex storylines that simulate entire battles, this action knows no bounds!

## Fields

<ActionsField/>

<EntryField name="MobName" required>
    The mob's name.
</EntryField>

<EntryField name="Level" required>
    The mob's level.
</EntryField>

<EntryField name="Spawn Location" required>
    Spawn Location, can use `~` to define player's relative location.
</EntryField>
