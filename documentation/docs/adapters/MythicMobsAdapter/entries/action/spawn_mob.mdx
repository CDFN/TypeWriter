# Spawn Mob Action

The `Spawn Mob Action` action spawn MythicMobs mobs to the world.

## How could this be used?

This action could be used in a plethora of scenarios. From simple quests requiring you to kill some spawned mobs, to complex storylines that simulate entire battles, this action knows no bounds!

## Fields

<ActionsField/>

<EntryField name="Mob" required>
    The mob's name.
</EntryField>

<EntryField name="Level" required>
    The mob's level.
</EntryField>

<EntryField name="UsePlayerLocation" required>
    Whether or not to use the player's position as the center when summoning.
</EntryField>

<EntryField name="RelativeLocation" required>
    If `UsePlayerLocation` is false, then it will be spawn location.
</EntryField>
